# Transjakarta Customer Segmentation Analysis

## 📌 Objective
Analyze passenger behavior to identify customer segments and support data-driven strategies for increasing public transportation usage.

## 📊 Dataset
- Total: 180,000+ passenger trip records  
- Features include:
  - Trip frequency  
  - Travel time  
  - Routes  
  - Usage patterns  

## ⚙️ Tools & Technologies
- Python (Pandas, Scikit-learn)
- Data Visualization (Matplotlib, Seaborn)
- Looker Studio (Dashboard)

## 🔍 Methodology
1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA) to identify usage patterns  
3. Customer segmentation using **K-Means Clustering**  
4. Determining optimal clusters using:
   - Elbow Method  
   - Silhouette Score  
5. Visualization and dashboard development using Looker Studio  

## 📈 Results
- Identified **3 main customer segments**:
  - High Value Users  
  - Occasional Users  
  - Low Frequency Users  

## 🔑 Key Insights
- High Value Users dominate peak hours and contribute most to ridership  
- Occasional Users show potential for increased usage with targeted incentives  
- Low Frequency Users are sensitive to accessibility and scheduling  

## 💡 Business Impact
- Enables **targeted marketing strategies** for each customer segment  
- Supports **service optimization** based on usage patterns  
- Provides a foundation for **increasing overall ridership**
